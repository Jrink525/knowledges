# 20 个 Claude Prompt，让你的 $20/月 订阅变成全能副手

> **来源：** [20 Claude Prompts that turn a $20 Subscription into a personal Assistant, Editor, Coach, and Analyst](https://x.com/anatolikopadze/status/2049492553133629950)
> **作者：** Anatoli Kopadze (@AnatoliKopadze)
> **日期：** 2026-04-29
> **数据：** 1204 ❤️ · 6590 🔖 · 2.44M 阅读

---

你每月付 $20 订阅 Claude。大多数人只用了它 10% 的能力。

问个问题。改写一段文字。总结一篇文档。这就是普通用户的 Claude 会话。

> 这不是在用 Claude。这是在浪费它。

Claude 可以是你的健身教练、你的语言老师、你的谈判训练师、你的财务分析师、你的编辑、你的策略师。不是"差不多"。是**字面上**可以。唯一的变量是你 prompt 的质量。

这 20 个 prompt 覆盖了真实生活中的真实场景。挑你需要的。跳过不需要的。把它们作为构建你自己 prompt 的基础。

**保存这篇文章。**

---

## 一、深度研究（Deep Research）

### 1️⃣ 多源综合（Multi-Source Synthesis）

别再分开读 5 篇文章了。把它们全部粘贴到一个对话里，让 Claude 蒸馏成一份单一的、解决完冲突的简报。它会捕捉每篇文章错过的东西。它会发现它们之间有分歧的地方。

- 你得到：3 分钟
- 不这样做：90 分钟阅读

```
请综合以下 [N] 篇文章的要点。
找出它们一致的地方，以及有分歧的地方。
对每个分歧点，给出各方的论据强度。
最终产出一个结构化的简报，适合 [XX] 场景使用。
```

### 2️⃣ 魔鬼代言人模式（Devil's Advocate Mode）

在你对任何想法、计划或决定做出承诺之前——让 Claude 尽可能用力地反驳它。不是客气地。不是带着条件。用最强的理由论证你为什么错。

它找到的漏洞，就是那些以后会害了你的漏洞。

```
请扮演魔鬼代言人。
对我的这个想法 [粘贴你的想法] 进行最强烈的反驳。
不要客气，不要妥协。
用三到五个独立的角度攻击它。
每个角度都要给出具体的致命问题。
```

### 3️⃣ 钢化任何立场（Steelman Any Position）

你不同意某事。在否定它之前，让 Claude 把它钢化——构建一个最好、最善意、最聪明的对立论证版本。不是为了改变你的想法。是为了理解聪明人在另一面**实际**相信什么。

> 这是你不再被 surprises 惊讶的方法。

```
请帮我钢化以下立场：[粘贴你不认同的观点]
构建这个论点最有力、最善意、最聪明的版本。
不要 strawman 它。
给出支持这个立场的 3-5 个最强论据，
以及每个论据背后的核心洞察。
```

### 4️⃣ 心智模型构建器（Mental Model Builder）

任何你不懂的话题——让 Claude 用一系列关联的心智模型来教你。不是事实。不是摘要。一个你**可以用来思考的框架**。

10 分钟后你就能给别人解释清楚，并应用到新的场景里。这就是"知道"和"理解"的区别。

```
请用心智模型的方式教我 [话题]。
不要给事实列表。不要给摘要。
构建 3-5 个关联的心智模型，
让我可以用它们来思考这个领域。
每个模型给一个现实世界的类比。
然后出一道测试题检验我的理解。
```

## 二、写作（Writing）

### 5️⃣ 风格模仿（Style Mimic）

粘贴 3 份你自己的写作样本。让 Claude 分析你的风格——节奏、词汇、句子长度、你怎么开头、你怎么收尾。然后给一个话题，让它用你的声音写。不是普通的 AI 输出。是**你的声音**。

结果比你对着空白页写出来的更接近你真正会写的东西。

```
先分析我提供的 3 篇写作样本的风格：
- 节奏（长短句比例、断句方式）
- 词汇偏好（正式/口语化、专业术语密度）
- 段落结构（怎么开头、怎么过渡、怎么收尾）
- 语气特点（自信/谨慎/幽默/严肃）

然后以我的风格写一篇关于 [话题] 的文字。
```

### 6️⃣ 残酷编辑（The Brutal Editor）

粘贴你的草稿。让 Claude **摧毁**它。不是"改进它"。不是"给建议"。摧毁它。找出每一个浪费读者时间的句子、每一个不配存在的想法、每一个可以删掉的词。

经过这个版本留存下来的，才是值得发表的。

```
请以最残酷的方式编辑以下草稿。
规则：
1. 砍掉任何废话
2. 标注每一个不精确的词
3. 删掉任何没有进度的段落
4. 告诉我原本应该怎么写，而不是只做标记
不要给我留情面。我要的是值得发表的版本。
```

### 7️⃣ 一稿五面（One Text, Five Audiences）

你写了一篇东西给一个人。让 Claude 同时为五个不同的受众重写它。相同的核心观点。完全不同的语气、词汇和框架。

- 一个 5 岁小孩
- 一个领域专家
- 一个怀疑论者
- 一个记者
- 你的 CEO

> 从这种折射中看到你自己的观点，比任何其他练习学到的东西都多。

```
请为以下五个不同的受众重写这段文字，保持核心观点不变：
1. 一个 5 岁的孩子
2. 这个领域的专家
3. 一个怀疑论者
4. 一个记者
5. 你的 CEO

原文：[粘贴你的文字]
```

### 8️⃣ 要点变文章（Bullets to Article）

你有粗糙的笔记。散落的要点。碎片。让 Claude 把它们转化为一篇完整的、有结构的文章。提供目标长度、目标受众和语气。

要点变成骨架。Claude 填充血肉。你把精力花在思考上，而不是排版上。

```
请将以下要点扩展成一篇完整的文章。
- 目标长度：[字数]
- 目标受众：[描述]
- 语气：[正式/口语化/幽默/严肃]

要点：
[粘贴你的要点笔记]
```

## 三、职业发展（Career）

### 9️⃣ 薪资谈判角色扮演（Salary Negotiation Roleplay）

大多数人在薪资谈判中损失数千美元，因为他们从未练习过。让 Claude 扮演你未来的雇主——给它背景信息、offer 内容、以及他们可能的反对点。实时谈判。

Claude 会回击。它会用他们会用的论证。等你真正走进谈判室，你已经练过十次了。

```
你现在是一家 [公司类型] 的招聘负责人。
我申请了 [职位] 岗位。我的背景是 [简述]。
我收到的 offer 是 [具体数字]。
请扮演你方角色，我要进行薪资谈判。
你要用真实的反对理由来推回我的要求。
不要让我轻松过关。
```

### 🔟 面试模拟器（Interview Simulator）

粘贴职位描述。让 Claude 扮演面试官。它会问该角色通常会出现的问题——行为题、技术题、情景题。

你回答。它给你打分：什么答得好，什么太模糊，你本应该说什么。

> 练三次，你走进任何面试都能保持冷静。

```
这是 [职位] 的 JD：
[粘贴 JD]
请扮演面试官，对我进行面试。
先问行为题，然后技术题，最后情景题。
每题我回答后，请给我打分（1-10）并告诉我：
- 什么说得好
- 什么太模糊
- 更好的回答应该是什么
```

### 1️⃣1️⃣ 简历放大器（Resume Multiplier）

你只有一份简历。但你申请的工作各不相同。粘贴你的简历和三份不同的职位描述。让 Claude 生成三个版本——每份都凸显对该岗位最重要的经历。

> 你的简历不再是"对谁都不特别合适"。

```
这是我的简历：[粘贴简历]
以下是三个不同的职位描述：
A: [粘贴 JD A]
B: [粘贴 JD B]
C: [粘贴 JD C]

请为每个岗位生成一份针对性优化的版本。
每份保持事实不变，但调整重点和措辞，
让最相关的经历优先出现。
```

### 1️⃣2️⃣ 困难对话准备（Difficult Conversation Prep）

你需要进行一次困难的对话。和你的经理。客户。联合创始人。

大多数人准备不足就直接上，说太多或说太少。向 Claude 描述情况，让它帮你规划对话：该说什么，他们可能会回什么，以及如何应对每种场景。

> 你不是在即兴发挥。你是在执行剧本。

```
我需要和 [对象身份] 进行一次困难的对话。
背景：[描述背景]
我需要达成：[你的目标]
他们的立场可能是：[预期他们的立场]
请帮我：
1. 规划整段对话的节奏
2. 给出开场白话术
3. 预测 3 种他们的回应方式及我的应对
4. 告诉我绝对不能说什么
```

### 1️⃣3️⃣ LinkedIn 个人简介重写（LinkedIn Bio Rewrite）

你的 LinkedIn 简介是三年前花 10 分钟写的。它不是太谦虚就是太模板化。粘贴它，告诉 Claude 你想吸引什么人以及为什么，要 5 个版本。不同角度。不同语气。

```
请重写我的 LinkedIn 简介。
现有版本：[粘贴现有简介]
我想吸引：[目标人群/机会类型]
我的核心卖点是：[你的独特价值]
请给 5 个版本，从不同角度切入。
每个版本附上简短的策略说明。
```

## 四、日常生活（Daily Life）

### 1️⃣4️⃣ 每周餐单 + 购物清单（Weekly Meal Plan with Shopping List）

告诉 Claude 你的饮食偏好、热量目标和你能做饭的时间。它会构建一个完整的 7 天计划（含早午晚餐），加上一份按超市分区整理的购物清单。

> 每天"吃什么"的选择疲劳——3 分钟消失一整周。

```
我每天需要 [卡路里] 大卡。
我不吃/偏好：[饮食限制/偏好]
我每天愿意花 [分钟] 做饭。
请给我一个 7 天的餐单（早+午+晚），
同时生成一份按超市分区整理的购物清单。
要有可操作性，不要需要找不到的食材。
```

### 1️⃣5️⃣ 法律文档翻译器（Legal Document Translator）

你收到了一份合同、TOS 更新、租赁协议或隐私政策。你一半不理解。粘贴它，让 Claude 翻译成大白话——你在同意什么，哪些条款有风险，签之前你应该问什么。

> 你不再在对重要文档一无所知的情况下签字。

```
请将以下法律文档翻译成普通人能懂的大白话。
我需要你告诉我：
1. 我实际上在同意什么（一句话总结）
2. 列表中的 3 个最高风险条款
3. 每个风险条款我该怎么做
4. 签之前我必须问的 3 个问题

文档：[粘贴]
```

### 1️⃣6️⃣ 个人财务分析（Personal Finance Analyzer）

粘贴你上个月的支出。让 Claude 分类、找规律，准确指出钱从哪里漏出去了。不是"少在外面吃"这种模糊建议。是**具体的**观察。

> "你在订阅上花了 340 美元，其中 4 个服务你本月用了不到一次。"
> 这种具体才是真正改变行为的东西。

```
我上个月的支出如下：
[粘贴你的支出记录]

请帮我：
1. 分类汇总（固定支出/餐饮/娱乐/订阅/交通等）
2. 找出 3 个最大的"钱漏"点
3. 给出可量化的改进建议（不要笼统的）
4. 对比我这个月和上个月的变化
```

### 1️⃣7️⃣ 旅行规划器（Travel Planner）

告诉 Claude 你去哪、待多久、预算多少、你是哪种旅行者。它会生成一份逐日行程，包括大多数游客不知道的东西——社区小店、时机技巧、需要提前预定的东西。

> 它不是 TripAdvisor 的复制。它是适合你的计划。

```
我要去 [目的地] [天数] 天。
预算 [金额]。
我的旅行风格：[独行/家庭/穷游/舒适/探险]
我的兴趣：[文化/美食/户外/购物/当地体验]
请给我逐日行程：
- 每天早中晚的安排
- 本地人才知道的推荐
- 需要提前预定的项目
- 省钱/省时间的技巧
```

## 五、学习（Learning）

### 1️⃣8️⃣ 费曼导师（Feynman Tutor）

费曼技巧：当你能用简单的话解释一个东西，你才真正懂了。让 Claude 用这个方法教你任何话题——从简单开始，确认你理解了才增加复杂度，用你已经知道的东西做类比。

> 它不是读。它是学。

```
请用费曼技巧教我 [话题]。
规则：
1. 从最简单的版本开始
2. 问问题确认我理解了才继续
3. 每步用我已经知道的东西做类比
4. 如果我有疑问，耐心拆解到我最薄弱的根基
5. 不要在概念没站稳之前跳入下一步
```

### 1️⃣9️⃣ 30 天课程生成器（30-Day Curriculum Builder）

你想学一项技能。大多数人开始了，在第二周被压垮，然后放弃——因为他们没有地图。

让 Claude 构建一份逐日的 30 天课程——学什么、按什么顺序、每天学多久、有检查点来确认你的进展。

> 技能不再是"我想学的东西"，它变成了一个有截止日期的项目。

```
我想在 30 天内学会 [技能]。
我的背景：[现有知识水平]
我每天可以投入 [分钟/小时]。
请生成：
- 按天划分的课程计划（30 天）
- 每阶段的学习目标
- 每 5 天的检查点测试
- 推荐的练习资源和工具
- Day 1 的具体学习内容
```

### 2️⃣0️⃣ 苏格拉底模式（Socratic Mode）

这是 Claude 最被低估的能力。与其要答案，不如让 Claude 通过**提问**来教你。

你告诉它话题。它问你问题。你回答。它在你的回答上构建，纠正你的空缺，让你通过自己的思考来理解。

> 被动阅读产生熟悉感。苏格拉底对话产生知识。
> 两者之间有区别，你会立刻感受到。

```
请用苏格拉底提问法教我 [话题]。
规则：
1. 不要直接给我答案
2. 先问我对这个话题已经知道什么
3. 通过提问引导我发现盲点
4. 当我卡住时，给我一个提示而不是答案
5. 最后用一段话总结我通过自己的推理学到了什么
```

## 总结

20 个技巧：

| 类别 | 技巧 |
|---|---|
| **深度研究** | 多源综合 · 魔鬼代言人 · 钢化立场 · 心智模型 |
| **写作** | 风格模仿 · 残酷编辑 · 一稿五面 · 要点变文章 |
| **职业** | 薪资谈判 · 面试模拟 · 简历定制 · 困难对话 · LinkedIn |
| **日常生活** | 餐单规划 · 法律翻译 · 财务分析 · 旅行规划 |
| **学习** | 费曼导师 · 30 天课程 · 苏格拉底模式 |

- 深度研究：比 90 分钟阅读更好
- 写作：经过残酷编辑还能活下来的版本
- 职业准备：让困难对话像排练过一样
- 生活助理：处理那些耗尽你精力的决策
- 导师：通过你自己的推理来教你

每一个都包含在你的 $20/月里。

> 大多数人会读这篇文章，收藏它，然后回到用 Claude 总结邮件。
>
> **不要做大多数人。**
