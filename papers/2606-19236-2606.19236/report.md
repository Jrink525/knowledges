# STARE: 基于惊奇度（Surprisal）引导的令牌级优势重加权策略熵稳定性

## 论文背景

### 基本信息
- **论文标题**: STARE: Surprisal-Guided Token-Level Advantage Reweighting for Policy Entropy Stability
- **作者**: Haipeng Luo, Qingfeng Sun, Songli Wu, Can Xu, Wenfeng Deng, Han Hu, Yansong Tang
- **机构**: 清华大学深圳国际研究生院 & 腾讯混元
- **论文编号**: arXiv:2606.19236
- **代码**: https://github.com/hp-luo/STARE

### 研究背景

强化学习与可验证奖励（RLVR）已成为大语言模型（LLM）后训练的主流范式，以 DeepSeek-R1、Qwen3、Kimi K1.5 为代表。在这些算法中，分组相对策略优化（GRPO）摒弃了价值网络，采用组归一化奖励作为优势估计的基线，已在数学推理、代码生成等领域取得显著成功，能诱导出长思维链推理和自我反思等涌现行为。

然而，随着训练步数的增加，GRPO 类算法普遍面临**策略熵坍塌**（policy entropy collapse）问题：熵值快速衰减，输出多样性消失，策略过早收敛，组内 rollout 同质化，最终限制了可训练的步数。这构成了长视界后训练的关键瓶颈。

## 核心问题与方法

### 已有方法的局限性

现有缓解熵坍塌的方法可归为三类：

1. **调整裁剪阈值**（如 DAPO 的 clip-higher）：保护低概率探索性令牌，但对熵的影响不对称且不可控，在 on-policy 场景下基本上不生效（比率接近 1）。

2. **轨迹级不对称加权**：对正负 rollout 赋予不同权重（如稀有正确 rollout 上加权、偏置负样本），提供的控制粒度较粗。

3. **熵感知优势重塑**：将令牌级熵耦合到优势中，但倾向于过度放大高熵令牌，引起振荡，且超参数敏感。

这些方法在一定程度上减缓了熵衰减，但都是在轨迹或样本粒度操作，缺乏对熵坍塌机制的机理理解。

### STARE 的理论分析

本文的核心贡献在于对 GRPO 下的令牌级熵动态进行了**一阶梯度分析**，揭示了熵坍塌的根本机制：

#### 令牌级熵变化定理（Theorem 3.1）

在 GRPO 的无裁剪区域，令 Ā 为当前位置的优势值，h 为沿 GRPO 策略梯度方向步长，则：

```
dH/dh|_{h=0} = -Ā · F(p)
```

其中 F(p) 是**熵敏感度函数**，衡量采样令牌的概率加权惊奇度偏差超过分布基线 S₂ 的部分。

#### 优势-惊奇度四象限分解（Four-Quadrant Decomposition, Corollary 3.3）

dH/dh 的符号由 (sign(Ā), 1[s_a < s*]) 决定：

| 象限 | 属性 | 熵效应 |
|------|------|--------|
| Q1: Ā > 0, s_a < s* | 强化低惊奇度令牌 | **降低熵** |
| Q2: Ā > 0, s_a > s* | 强化高惊奇度令牌 | **增加熵** |
| Q3: Ā < 0, s_a < s* | 抑制低惊奇度令牌 | **增加熵** |
| Q4: Ā < 0, s_a > s* | 抑制高惊奇度令牌 | **降低熵** |

关键不对称性：由于 rollout 从策略 p_θ 采样，低惊奇度令牌被更频繁地采样。在正优势轨迹中，**熵降低的令牌（低惊奇度）构成统计多数**，而能增加熵的稀有高惊奇度令牌贡献有限，导致 GRPO 系统性驱动策略趋向集中化。

#### 近临界性质（Near-Criticality, Corollary 3.6）

熵中性恒等式 (Theorem 3.4) 表明 E_a~p[F(a)] = 0。通过令牌级优势重加权，临界权重 W* = 1 + L/G。当序列长度 T 和 batch size 足够大时：

```
W* - 1 = L/G = O(T^{-1})
```

这意味着：**一个微小的令牌级权重扰动就足以翻转熵演化的方向**，且超过临界阈值后，权重具体值主要控制熵变化的幅度而非方向。这一理论洞见直接驱动了算法设计。

## STARE 算法

### 算法架构

STARE 是在 GRPO 裁剪代理目标内部运作的最小侵入式令牌级信用再平衡机制，包含三个核心组件：

#### 1. 基于高惊奇度分位数的熵关键令牌划分

不直接计算每个位置的理论临界阈值 s*，而是采用批内惊奇度分位数代理：
- 在正优势令牌集 T⁺ 和负优势令牌集 T⁻ 内分别按惊奇度降序排序
- 选择前 P% 的令牌构成熵关键集 L⁺ 和 L⁻
- 缺省 P% = 10%

#### 2. 优势条件化令牌级信用再平衡

在 GRPO 目标中引入正令牌级权重 w_{i,t} > 0：

```
J_STARE(θ) = (1/N) Σ_{i,t} w_{i,t} · min(r_{i,t}(θ)Ā_i, clip(r_{i,t}(θ), 1-ε, 1+ε)Ā_i)
```

**变体 I（O1 — 单边熵放大）**：
- 对 L⁺ 中的令牌施加权重 W > 1（缺省 1.1）
- 所有其他令牌权重为 1

**变体 II（C2 — 双边熵调控）**：
- 对 L⁺ 施加 W > 1
- 对 L⁻ 施加 M < 1（缺省 0.9）
- 所有其他令牌权重为 1

#### 3. 目标熵闭环门控

为避免开环重加权导致的过度探索，引入目标熵 H_tgt 和闭环门控信号：

```
g_k = 1[Ĥ_k < H_tgt]
```

当批次平均熵 Ĥ_k 低于 H_tgt = 0.3 时激活重加权；当熵恢复后自动恢复为标准 GRPO。这使得熵在目标附近形成有界振荡，实现闭环稳定调控。

#### 自适应权重调度（可选扩展）

支持基于目标熵信号的自适应更新：

```
W_{k+1} = clip(W_k + α·sgn(H_tgt - Ĥ_k), [1, W_max])
M_{k+1} = clip(M_k - α·sgn(H_tgt - Ĥ_k), [M_min, 1])
```

缺省 α = 0.01, W_max = 1.5, M_min = 0.5。

## 实验分析

### 实验设置

**模型与场景**：

| 场景 | 模型 | 最大解码长度 |
|------|------|-------------|
| Short CoT | Qwen2.5-Math-7B-Base | 4k |
| Short CoT | Qwen2.5-14B-Instruct | 8k |
| Short CoT | Qwen2.5-32B-Base | 8k |
| Long CoT | DeepSeek-R1-Distill-Qwen-1.5B | 16k |
| Long CoT | Qwen3-8B-Base | 16k |
| Tool-Use | Qwen2.5-7B-Base + Retool 2K SFT | 8k |

**训练参数**：
- 学习率：1×10⁻⁶
- Batch size：64 samples × 8 rollouts
- On-policy 更新，每 batch 单梯度步
- Top-p = 1.0，Temperature = 1.0
- 训练语料：100k 样本，来自 DeepScaler、Skywork-o1、Polaris、DAPO

**STARE 缺省配置**：W = 1.1, M = 0.9, H_tgt = 0.3, P% = 10%（变体 O1 + 批次级闭环门控 + 固定权重）

**评估基准**：AIME24, AIME25, AMC23, MATH-500, Minerva Math, OlympiadBench（Top-p=0.95, T=0.7, AIME 评估32次取平均）

### 训练动态分析

- **熵稳定性**：GRPO-ds 在 0-1000 步内熵值趋近于零（熵坍塌），AIME 准确率在约 1000 步达到峰值后饱和。STARE 将熵稳定在目标值 0.3 附近，准确率持续提升至 5000 步。

- **探索-利用平衡**：STARE 的 Pass@32 全程超过 GRPO-ds，保持输出多样性并缓解模式寻求。GRPO-ds 的奖励和完全解题率早早进入平台期，而 STARE 持续增长。

- **反思令牌涌现**：词云分析显示 STARE 重加权的令牌集中在不确定性和自我修正词汇（should be, but, instead, verification 等），确认了其有效识别了具探索语义的罕见分叉令牌。

- **跨规模、跨场景泛化**：在 1.5B→32B 所有模型规模和 Short CoT、Long CoT、Tool-Use 三个场景中，STARE 一致保持熵稳定和准确率持续提升。

### 消融研究

1. **重加权因子 W 的影响**：
   - 打开闭环门控：W=1.01 即可缓解熵衰减，W≥1.05 实现稳定增长，W≥2.0 触发发散
   - 闭环门控：所有 W ∈ [1.05, 1.5] 均能将熵引导到目标带，极大降低对 W 的敏感性

2. **分位数比例 P 的影响**：
   - P ∈ [5%, 20%] 时熵稳定在目标带内
   - 即使 P=40%，熵仍被限制在 [0.1, 0.2] 区间
   - 表明 STARE 具有广泛的操作窗口

3. **高惊奇度分位数代理验证**：
   - 所选令牌落入理论熵增区域的比例从约 60% 稳步升至约 95%
   - L⁺q 的累积净熵贡献保持正单调增长
   - 验证了代理与实际临界阈值的高度一致性

4. **单极性和组合操作**：
   - 8 种变体均有效缓解熵坍塌并显著超越 GRPO-ds
   - O1（放大 L⁺）和 C2（同时放大 L⁺、衰减 L⁻）表现最佳

5. **其他消融**：固定 vs 自适应加权、目标熵阈值、门控粒度、off-policy 训练、固定阈值重加权等均进行了详细分析（见附录）。

## 结论与展望

### 主要贡献

1. **理论层面**：首次从一阶梯度角度分析了 GRPO 中令牌级熵动态，揭示了优势-惊奇度四象限结构和近临界性质——一个微小的令牌级权重扰动就足以翻转熵演化方向。

2. **算法层面**：提出 STARE——基于惊奇度分位数的令牌级优势重加权机制，耦合目标熵闭环门控，仅通过对 GRPO 目标的最小修改实现了稳定的策略熵调控。

3. **实证层面**：在 1.5B 到 32B 的多模型规模和 Short CoT、Long CoT、Tool-Use 三个场景中验证，STARE 稳定维持数千步 RL 训练，并在 AIME24/25 上超越 DAPO 等基线 4-8%。

### 局限性

- 当前分析基于一阶梯度近似，未考虑高阶效应和裁剪区域的影响
- 目标熵 H_tgt 仍需人工设定，虽然下游性能对其不敏感
- 实验主要覆盖数学推理场景，对其他领域的泛化性尚需验证
- 32B 以上更大模型的扩展性有待进一步研究

### 未来方向

1. 将惊奇度分位数代理扩展到自适应的、无需人工设定阈值的方法
2. 探索 STARE 在代码生成、科学推理等其他领域的应用
3. 结合更精细的令牌级信用分配机制（如逐令牌奖励分解）
4. 将四象限分析框架推广到 PPO 等其他 RLHF 算法
5. 研究熵调控与模型泛化能力之间的深层关系
